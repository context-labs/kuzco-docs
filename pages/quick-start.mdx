# Quick Start

<Callout type="warning">
  This documentation is a work in progress. Join Discord for more information.
</Callout>

Kuzco can run a macOS Desktop App or a Linux CLI. The following instructions will guide you through the installation process for each platform.

## Kuzco macOS Desktop App

1. Download the Kuzco macOS desktop app from the official website [here](https://kuzco.xyz/download).
2. Install the app by dragging it to your Applications folder.
3. Launch the Kuzco app and follow the on-screen instructions to set up your node and connect to the network.
4. Once connected, you can start contributing your idle compute power and explore the app's features.

## Kuzco Linux CLI Tool

1. Open a terminal on your Linux machine.
2. Run the following command to install the Kuzco CLI tool:

   ```
   curl -fsSL https://kuzco.xyz/install.sh | sh
   ```

3. After installation, run the following command to initialize your Kuzco node:

   ```
   kuzco init
   ```

4. If this command fails you may need to re-run as `sudo`.
5. Follow the prompts to configure your node and connect to the Kuzco network.
6. Once connected, you can start contributing your idle compute power and use the CLI tool to interact with the network.

For more detailed instructions and advanced configuration options, refer to the respective sections of the documentation.
